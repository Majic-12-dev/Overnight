# Batch 3: Innovative Features

## Tool 1: Data Format Converter (productivity)
Convert between CSV, TSV, JSON, XML, YAML formats with column mapping, live preview, 
copy-to-clipboard, and file download support.

## Tool 2: Image Slicer (image)  
Split images into equal grid pieces (NxM grid), export as individual files or ZIP.
Useful for web tiles, sprite sheets, and social media carousel preparation.

## Tool 3: Meta Tag Generator (text)
Generate complete HTML meta tags including:
- Standard meta (title, description, keywords, author)
- Open Graph tags (og:title, og:description, og:image, og:url, og:type)
- Twitter Card tags
- JSON-LD structured data
Output: Copy HTML or download as .html file.

## Tool 4: Color Contrast Checker (productivity)
WCAG 2.1 contrast ratio checker. Input two colors, get:
- Contrast ratio
- AA/AAA pass/fail for normal and large text
- Suggestions for passing colors

## Dependencies needed:
- js-yaml (for YAML handling in data converter) - check if already present
